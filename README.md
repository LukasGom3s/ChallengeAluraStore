# ChallengeAluraStore
Vamos decidir qual loja da rede Alura Store vender para iniciar um novo empreendimento. Para isso, vamos analisar dados de vendas, desempenho e avaliações das 4 lojas fictícias da Alura Store. O objetivo é identificar a loja com menor eficiência e apresentar uma recomendação final baseada nos dados.
